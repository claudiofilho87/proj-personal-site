Projects

Desenvolvimento de um site para controle e execução de robôs da IQ Option em Python.

O projeto consistiu na criação de um site simples em design, mas com sistema completo de autenticação (login, cadastro e recuperação de senha) e uma página inicial para configurar e executar até quatro robôs simultaneamente. Desenvolvido com Laravel/PHP e MySQL, inclui uma API em Python com Flask para acionamento dos robôs e gerenciamento de execuções via Celery e Redis. A limitação de quatro robôs simultâneos foi definida com base nos recursos do servidor, mas podendo ser ajustada. O deploy foi realizado em uma VPS na Hostinger usando GitHub e Docker para gerenciar os serviços em containers.
